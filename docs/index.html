Mission Statement

The Panther Web Development Group, also known as PantherWeb, aims to provide a collaborative atmosphere to Georgia State University students of all majors and backgrounds interested in web development. We encourage students to learn about web development, create websites, network, understand the importance of UI/UX design, and apply the concepts of web development through projects.

We allow students to create websites (and mobile applications) using languages and frameworks used in web development, such as HTML, CSS, JavaScript, React, Vue, Angular, PHP, and more.
